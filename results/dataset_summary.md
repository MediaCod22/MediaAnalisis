## Использование данных

### Формат записи (JSON)

```json
{
  "id": "IT_001",
  "url": "https://example.com/article",
  "name": "Заголовок материала",
  "snippet": "Краткое описание...",
  "host_name": "example.com",
  "rank": 0,
  "_sector": "IT",
  "_source": "google",
  "_file": "IT_google_1.json",
  "assessment": {
    "P1_relevance": 9,
    "P2_scientific": 5,
    "P3_currency": 8,
    "P4_credibility": 5,
    "P5_practical": 7,
    "IKM": 6.8,
    "quality_class": "средний",
    "source_type": "blog",
    "has_doi": false
  }
}
```

### Формат записи (CSV)

| Колонка | Описание |
|---------|----------|
| ID | Уникальный идентификатор (СФЕРА_НОМЕР) |
| URL | Ссылка на материал |
| Источник_сбора | yandex/google/scholar/elibrary |
| Позиция_в_выдаче | Позиция в результатах поиска |
| Заголовок | Название материала |
| Домен | Хост источника |
| P1-P5 | Оценки по параметрам ИКМ |
| ИКМ | Итоговый индекс качества |
| Класс_качества | высокий/средний/низкий |
| Тип_источника | academic/industry/media/blog |
| DOI | Наличие DOI (Да/Нет) |
| Год | Год публикации |
