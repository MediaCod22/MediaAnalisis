# МЕТОДОЛОГИЯ СБОРА И АНАЛИЗА ДАННЫХ

## 1. ТЕОРЕТИЧЕСКИЕ ОСНОВАНИЯ

### 1.1. Концептуальная рамка исследования

Настоящее исследование базируется на междисциплинарном подходе, объединяющем:

1. **Информационный менеджмент** — анализ роли информации в процессах принятия решений
2. **Медиаэкономику** — изучение влияния медиаданных на экономические процессы
3. **OSINT (Open Source Intelligence)** — методология работы с открытыми источниками
4. **Систематический обзор** — академический стандарт PRISMA для обобщения источников

### 1.2. Определение ключевых понятий

| Термин | Определение |
|--------|-------------|
| **Медиаориентированный материал** | Любой контент в открытых источниках (статья, отчёт, аналитическая записка, пост), содержащий информацию, релевантную для принятия управленческих решений |
| **Медиачувствительность (IMS)** | Количественный показатель зависимости отрасли от медиаданных для принятия решений. Рассчитывается как взвешенная сумма: количество материалов × средний ИКМ × коэффициент отрасли |
| **ИКМ (Индекс Качества Материала)** | Комплексная оценка качества источника по 5 параметрам (P1-P5) по 10-балльной шкале |
| **Отраслевой кластер** | Группа материалов, относящихся к одной сфере экономики (IT, финансы, маркетинг, медицина, инженерия, образование) |

### 1.3. Гипотеза исследования

**H1:** Степень влияния медиаматериалов на процессы принятия управленческих решений существенно различается по отраслям и может быть количественно измерена через Индекс Медиачувствительности (IMS).

**Операционализация:**
- Зависимая переменная: IMS (непрерывная, 0-100)
- Независимая переменная: Отрасль (категориальная, 7 уровней)
- Контрольные переменные: ИКМ, тип источника, год публикации

## 2. МЕТОДОЛОГИЯ PRISMA

### 2.1. Обоснование выбора метода

Методология PRISMA (Preferred Reporting Items for Systematic Reviews and Meta-Analyses) выбрана по следующим причинам:

1. **Академический стандарт** — широко признана в научном сообществе, соответствует требованиям ВАК
2. **Воспроизводимость** — чётко документируемый процесс отбора источников
3. **Прозрачность** — каждый этап может быть проверен и воспроизведён
4. **Качество** — обеспечивает системный подход к оценке источников

## 3. ИСТОЧНИКИ ДАННЫХ

### 3.1. Выбор источников

| Источник | Тип | Обоснование включения |
|----------|-----|----------------------|
| **Яндекс** | Поисковая система | Основной поисковик Рунета, охват русскоязычного сегмента |
| **Google** | Поисковая система | Глобальный поисковик, охват англоязычных источников |
| **Google Scholar** | Академический поиск | Индексация научных публикаций, DOI |
| **eLibrary/РИНЦ** | Научная база | Российская научная электронная библиотека |

### 3.2. Стратегия сбора

**Глубина поиска:** Первый экран выдачи (первые 10 результатов)

**Обоснование:**
- Первый экран содержит наиболее релевантные результаты по алгоритмам поисковых систем
- Стандартная практика для OSINT-исследований
- Обеспечивает воспроизводимость методологии
- Позволяет избежать "шума" из нерелевантных материалов

**Паузы между запросами:** 3-5 секунд

**Обоснование:**
- Предотвращение блокировок со стороны поисковых систем
- Соблюдение принципов этичного сбора данных
- Имитация поведения человека-исследователя

### 3.3. Временной диапазон

**Основной массив:** 2020-2024 годы

**Обоснование:**
- Актуальность данных для современного состояния отраслей
- Учёт влияния пандемии COVID-19 на цифровизацию
- Отражение последних трендов в медиааналитике

## 5. СИСТЕМА ОЦЕНКИ КАЧЕСТВА (ИКМ)

### 5.1. Описание параметров

Индекс Качества Материала (ИКМ) — комплексная метрика, оценивающая пригодность источника для использования в процессах принятия решений.

#### P1: Релевантность (Relevance)

**Суть:** Степень соответствия материала теме исследования

**Алгоритм расчёта:**
```python
keyword_count = count_keywords_in_text(material)
P1 = min(10, 5 + keyword_count)
```

**Шкала:**
- 10 — Материал полностью посвящён теме
- 7-9 — Высокая релевантность
- 5-6 — Умеренная релевантность
- <5 — Низкая релевантность

#### P2: Научность (Scientific Rigor)

**Суть:** Принадлежность к академическим или экспертным источникам

**Алгоритм расчёта:**
```python
if domain in ACADEMIC_DOMAINS:
    P2 = 9  # ScienceDirect, Springer, Nature, CyberLeninka
elif domain in INDUSTRY_DOMAINS:
    P2 = 7  # McKinsey, Gartner, Deloitte
elif domain in MEDIA_DOMAINS:
    P2 = 5  # Forbes, HBR, Medium
else:
    P2 = 5  # Блоги, корпоративные сайты
```

**Академические домены:**
- sciencedirect.com, springer.com, wiley.com, nature.com
- frontiersin.org, mdpi.com, tandfonline.com
- researchgate.net, semanticscholar.org
- cyberleninka.ru, elibrary.ru

**Отраслевые домены:**
- mckinsey.com, gartner.com, deloitte.com
- ibm.com, microsoft.com
- harvard.edu, hbr.org

#### P3: Актуальность (Currency)

**Суть:** Свежесть данных, наличие актуальной информации

**Алгоритм расчёта:**
```python
year = extract_year_from_text(material)
if year >= 2025: P3 = 10
elif year >= 2024: P3 = 9
elif year >= 2023: P3 = 8
elif year >= 2022: P3 = 7
elif year >= 2021: P3 = 6
elif year >= 2020: P3 = 5
else: P3 = 4
```

#### P4: Достоверность (Credibility)

**Суть:** Авторитетность источника и возможность верификации

**Алгоритм:**
- Аналогичен P2 (репутация домена)
- Дополнительный бонус за наличие DOI

#### P5: Практическая ценность (Practical Value)

**Суть:** Применимость материала в реальных сценариях принятия решений

**Алгоритм расчёта:**
```python
practical_keywords = ["стратеги", "strategy", "внедрен", "implementation",
                     "практик", "practice", "метод", "method"]
practical_count = count_keywords(material, practical_keywords)
P5 = min(10, 5 + practical_count)
```

### 5.2. Формула итогового ИКМ

```
ИКМ = (P1 + P2 + P3 + P4 + P5) / 5
```

### 5.3. Классификация материалов

| Класс | Диапазон ИКМ | Рекомендация |
|-------|--------------|--------------|
| **Высокое качество** | ≥ 8.0 | Приоритетный анализ, включение в обзор |
| **Среднее качество** | 6.0 – 7.9 | Базовый анализ, контекстуальное использование |
| **Низкое качество** | < 6.0 | Исключается из углублённого анализа |

## 6. ПОЛИТИКО-ПРАВОВОЙ ФИЛЬТР

### 6.1. Обязательные исключения

В соответствии с требованиями к академическим публикациям в РФ, из выборки исключены материалы:

1. **Политически окрашенные:**
   - Критика Российской Федерации
   - Оценка политических процессов, включая СВО
   - Обсуждение спорных территорий (Крым, Донбасс)

2. **От нежелательных субъектов:**
   - Материалы от организаций, признанных нежелательными в РФ
   - Авторы со статусом иностранного агента
   - Экстремистские организации

3. **Украинские источники:**
   - Полное исключение украинских изданий и авторов
   - Исключение: советские классические работы (с двойной проверкой)

4. **Запрещённые темы:**
   - Пропаганда ЛГБТ
   - Пропаганда смены пола
   - Иные темы, запрещённые законодательством РФ

### 6.2. Механизм фильтрации

```python
EXCLUDED_DOMAINS = [
    "bellingcat.com",
    "meduza.io",
    # ... другие домены
]

def political_filter(material):
    domain = material.get("host_name", "").lower()
    if domain in EXCLUDED_DOMAINS:
        return False
    return True
```
