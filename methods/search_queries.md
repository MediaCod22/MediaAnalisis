# ПОИСКОВЫЕ ЗАПРОСЫ

## Методика формирования запросов

Каждый запрос формировался по принципу: **[Предметная область] + [Ключевое действие] + [Контекст]**.

- **Предметная область:** термины, специфичные для отрасли (IT, финансы, маркетинг и т.д.)
- **Ключевое действие:** "принятие решений", "decision making", "аналитика", "analytics"
- **Контекст:** дополнительные уточняющие термины
