# ГИПОТЕЗЫ ИССЛЕДОВАНИЯ

## Основная гипотеза

**H1: Степень влияния медиаматериалов на процессы принятия управленческих решений существенно различается по отраслям и может быть количественно измерена через Индекс Медиачувствительности (IMS).**

---

## Операционализация переменных

### Зависимая переменная

**IMS (Index of Media Sensitivity) — Индекс Медиачувствительности**

Количественный показатель, отражающий степень зависимости отрасли от медиаданных для принятия решений.

**Формула расчёта:**

```
IMS = (N × IKM_avg × K_sector) / N_max × 10

где:
- N — количество материалов в отрасли
- IKM_avg — средний ИКМ по отрасли
- K_sector — коэффициент цифровизации отрасли
- N_max — максимальное количество материалов в выборке
```

**Шкала IMS:**
- 8-10: Высокая медиачувствительность
- 5-7.9: Средняя медиачувствительность
- 0-4.9: Низкая медиачувствительность

### Независимая переменная

**Отрасль (категориальная переменная, 7 уровней)**

| Уровень | Описание | Код |
|---------|----------|-----|
| IT | Информационные технологии | 1 |
| Finance | Финансы и банковское дело | 2 |
| Marketing | Маркетинг и реклама | 3 |
| Medicine | Здравоохранение и медицина | 4 |
| Engineering | Инженерия и производство | 5 |
| Education | Образование | 6 |
| Bubbles | Кейс: технологические пузыри | 7 |

### Контрольные переменные

| Переменная | Тип | Описание |
|------------|-----|----------|
| ИКМ | Непрерывная | Среднее качество материалов в отрасли |
| Тип источника | Категориальная | Академический / Отраслевой / СМИ / Блог |
| Год публикации | Категориальная | 2020-2024 |
| Язык | Категориальная | Русский / Английский |

## Производные гипотезы

### H1.1: IT-сектор — лидер по медиачувствительности

**Формулировка:** IT-сектор демонстрирует наивысший Индекс Медиачувствительности среди всех отраслей.

**Обоснование:**
- IT-компании первыми внедряют технологии медиааналитики
- Цифровая природа отрасли обеспечивает прямой доступ к данным
- Культура data-driven решений наиболее развита в IT

**Критерий проверки:** IMS(IT) > IMS(другие отрасли) при p < 0.05

### H1.2: Финансы — средняя медиачувствительность при высоких требованиях к качеству

**Формулировка:** Финансовый сектор характеризуется средним уровнем медиачувствительности при статистически значимо более высоких требованиях к достоверности источников.

**Обоснование:**
- Оперативная информация критична для инвестиционных решений
- Консервативная культура требует верификации данных
- Регуляторные требования к источникам

**Критерий проверки:**
- IMS(Finance) ≈ среднее по выборке
- P4(credibility) для Finance > P4(другие отрасли) при p < 0.05

### H1.3: Медицина и образование — низкая медиачувствительность

**Формулировка:** Медицина и образование — отрасли с низкой медиачувствительностью, где решения опираются преимущественно на экспертное знание и доказательную базу, а не на медиаданные.

**Обоснование:**
- Профессиональные стандарты требуют доказательной медицины
- Решения основаны на клинических исследованиях, а не медиатрендах
- Образование регулируется государственными стандартами

**Критерий проверки:**
- IMS(Medicine) < среднее по выборке
- IMS(Education) < среднее по выборке
- Доля академических источников в Medicine, Education > других отраслей

### H1.4: Корреляция цифровизации и медиачувствительности

**Формулировка:** Существует положительная корреляция между уровнем цифровизации отрасли и её Индексом Медиачувствительности.

**Обоснование:**
- Цифровые каналы формируют первичный поток данных для анализа
- Автоматизация процессов повышает зависимость от данных
- Цифровая зрелость отрасли = способность использовать медиаданные

**Критерий проверки:** r(Цифровизация, IMS) > 0.6 при p < 0.05

## Кейс-гипотеза (H2)

**H2: Аномалия "Ethernet-пузырь"**

**Формулировка:** Исторический кейс "Ethernet-пузырь" демонстрирует ограниченность медиааналитики: медиапубликации не коррелировали с фундаментальными показателями отрасли.

**Обоснование:**
- В период пузыря медиа создавали ажиотаж, не отражающий реальность
- После краха медиаданные не предсказали падения
- Кейс показывает границы применимости медиааналитики

**Критерий проверки:**
- Низкая корреляция между медиаактивностью и реальными показателями
- Отсутствие предиктивной силы медиаданных в период пузыря

## Методы проверки гипотез

| Гипотеза | Статистический метод | Уровень значимости |
|----------|---------------------|-------------------|
| H1 | ANOVA (межгрупповое сравнение) | p < 0.05 |
| H1.1 | Post-hoc тесты (Tukey HSD) | p < 0.05 |
| H1.2 | t-критерий Стьюдента | p < 0.05 |
| H1.3 | Сравнение долей (χ²) | p < 0.05 |
| H1.4 | Корреляционный анализ (Пирсон) | p < 0.05 |
| H2 | Анализ временных рядов | p < 0.10 |

## Ограничения исследования

1. **Смещение выборки:** Первый экран выдачи может не отражать весь спектр материалов
2. **Языковой барьер:** Преимущественно русскоязычные и англоязычные источники
3. **Временные рамки:** Данные 2020-2024 могут не отражать долгосрочные тренды
4. **Качество оценки:** ИКМ основан на автоматическом анализе, возможны ошибки классификации
