## Аннотация

Настоящий репозиторий содержит данные и методологию систематического обзора медиаориентированных материалов, используемых для аналитики и принятия решений в различных отраслях. Исследование проведено по методологии **PRISMA** (Preferred Reporting Items for Systematic Reviews and Meta-Analyses) с применением оригинальной методики оценки качества источников **ИКМ** (Индекс Качества Материала).

## Гипотеза исследования

**Основная гипотеза (H1):** Степень влияния медиаматериалов на процессы принятия управленческих решений существенно различается по отраслям и может быть количественно измерена через Индекс Медиачувствительности (IMS — Index of Media Sensitivity).

**Производные гипотезы:**

| Код | Гипотеза | Обоснование |
|-----|----------|-------------|
| H1.1 | IT-сектор демонстрирует наивысшую медиачувствительность ввиду цифровой природы отрасли | IT-компании первыми внедряют технологии медиааналитики |
| H1.2 | Финансовый сектор характеризуется средней медиачувствительностью при высоких требованиях к достоверности | Зависимость от оперативной информации при консервативной культуре |
| H1.3 | Медицина и образование — отрасли с низкой медиачувствительностью, где решения опираются на экспертное знание, а не медиаданные | Профессиональные стандарты требуют доказательной базы |
| H1.4 | Существует прямая корреляция между цифровизацией отрасли и её медиачувствительностью | Цифровые каналы формируют первичный поток данных для анализа |

**Кейс-исследование (H2):** Аномалия «Ethernet-пузырь» — исторический прецедент, когда медиапубликации не коррелировали с фундаментальными показателями отрасли, демонстрируя ограниченность медиааналитики.

## Цель исследования

Выявить, систематизировать и оценить качество медиаориентированных источников информации, применяемых в процессах принятия решений в шести ключевых отраслях экономики.

## Задачи исследования

1. **Сбор данных:** Провести систематический поиск медиаматериалов по методологии PRISMA в четырёх источниках (Яндекс, Google, Google Scholar, eLibrary).
2. **Дедупликация:** Устранить повторяющиеся источники для формирования уникальной выборки.
3. **Оценка качества:** Рассчитать Индекс Качества Материала (ИКМ) для каждого источника по пяти параметрам.
4. **Кластеризация:** Распределить материалы по отраслям и провести сравнительный анализ.
5. **Валидация гипотезы:** Проверить корреляцию между уровнем цифровизации отрасли и её медиачувствительностью.

## Воспроизводимость

Все шаги исследования документированы и могут быть воспроизведены:

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сбор данных: `python methods/code/prisma_collector.py`
3. Обработайте данные: `python methods/code/prisma_processor.py`
4. Рассчитайте ИКМ: `python methods/code/ikm_calculator.py`
